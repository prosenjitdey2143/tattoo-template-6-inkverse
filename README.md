# INKVERSE | Ultra-Premium Tattoo Studio Template

![INKVERSE Banner](https://images.unsplash.com/photo-1598331668826-20cecc596b86?auto=format&fit=crop&q=80&w=1600)

**INKVERSE** is a high-fidelity, dark luxury, and cyberpunk-inspired website template designed for premium tattoo studios and avant-garde artists. Built with a focus on cinematic motion and high-end UI/UX, it delivers an Awwwards-level experience.

## ✨ Key Features

- 🌑 **Dark Luxury Aesthetic:** Premium cyberpunk-inspired UI with glassmorphism and ambient glow effects.
- 🚀 **Buttery Smooth Motion:** Integrated with **Lenis Scroll** for high-performance smooth scrolling.
- 🎯 **Advanced Interactions:** Custom lerp-based cursor and magnetic micro-interactions using **Anime.js**.
- 📽️ **Cinematic Video Player:** Fully custom-controlled video showcase with dynamic swapping logic.
- 📱 **Fully Responsive:** Optimized for all devices, from ultra-wide monitors to mobile screens.
- 🔍 **SEO & Accessibility Ready:** Semantic HTML5, ARIA labels, and JSON-LD Structured Data for search engines.
- 🔄 **Page Transitions:** Seamless navigation experience powered by **Barba.js**.

## 🛠️ Technology Stack

- **Core:** HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Animations:** [Anime.js](https://animejs.com/)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Sliders:** [Swiper.js](https://swiperjs.com/)
- **Transitions:** [Barba.js](https://barba.js.org/)
- **Typography:** Clash Display & General Sans (Fontshare)

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/prosenjitdey2143/tattoo-template-6-inkverse.git
   ```
2. **Open `index.html`** in your browser or use a Live Server.
3. **Customize:** Update the text, images in the `/assets` folder, and the WhatsApp number in `main.js`.
